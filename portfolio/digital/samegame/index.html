SameGame

Python, Tkinter

This was the final homework assignment for 15-112 Fundamentals of Programming. I implemented the classic same game in Python. I designed a flood-fill algorithm to create the select and remove part of the game and to keep score. This assignment was challenging algorithm-wise and we also learned to use FileIO to save a high score. I wanted my game to have a pastel theme so its gentle on the eyes. I was inpsired by flat design and I kept it very minimal and simple. The timer limits how much time the user has when selecting pieces to remove and the user can select the timing, the amount of colors, and the size of the game board.
